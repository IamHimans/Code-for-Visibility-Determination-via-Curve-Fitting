# Code-for-Visibility-Determination-via-Curve-Fitting
The present repository contains the code for calculating the visibility using curve fitting method and fitting the lab observed data of  visibility from a Michelson Interferometer Setup.

## Features
- Extracts the intensity variation of a given fringe along the pixel in x-direction

## How to Run
1. Install required libraries:
   pip install numpy matplotlib scipy

2. Run:
   python main_script.py
